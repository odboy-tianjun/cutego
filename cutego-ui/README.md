#### 安装教程

```
# 进入项目目录
cd cutego-ui

# 安装依赖，生成node_modules文件
npm install --registry https://registry.npmmirror.com --save
```

#### 使用说明

```
# 启动服务(使用 Vue 运行 Web 工程)
npm run dev

# 浏览器访问
http://localhost:16379

# 默认账号密码
用户名  admin   密码  123456
```

#### 发布

```
# 构建测试环境
npm run build:stage

# 构建生产环境(打包Web工程，生成dist文件)
npm run build:prod
```
